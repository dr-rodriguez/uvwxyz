Welcome to uvwxyz's documentation!
==================================

This documentation describes a pair of functions to compute stellar XYZ and UVW. This package requires numpy and astropy to be installed.

To install **uvwxyz**, do::

    pip install uvwxyz

You can then import the individual functions as follows::

    from uvwxyz import uvw, xyz
